This plugin allows you to accept online payments by SPS Clictopay SMT in WooComerce.

== Description ==

WooCommerce SMT ClicToPay is a simple and powerful plugin that enables your WooCommerce store to accept online payments via the SPS ClictoPay SMT payment gateway. This seamless integration allows your customers to make secure credit card payments directly on your store.

== Installation ==

1. Upload `woocommerce-smt-clictopay` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Settings > Payments tab to configure the plugin options.
4. Select 'ClicToPay SMT' to configure the settings. If you do not see 'ClicToPay SMT' in the list at the top of the screen, make sure you have activated the plugin through the 'Plugins' menu in WordPress.
5. Enter your ClicToPay API credentials and adjust any settings to fit your needs.

== Frequently Asked Questions ==

= Does this plugin support recurring payments? =

Currently, the WooCommerce SMT ClicToPay plugin does not support recurring payments.

= Is SSL required to use this plugin? =

Yes, an SSL certificate must be installed on your site to use WooCommerce SMT ClicToPay.
